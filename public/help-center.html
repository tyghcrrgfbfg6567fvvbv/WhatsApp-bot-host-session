Overview

What is WhatsApp Bot Manager?

WhatsApp Bot Manager is a powerful multi-session WhatsApp bot platform that allows you to manage multiple WhatsApp accounts from a single web interface. Built on Node.js and the Baileys library, it provides a comprehensive solution for automating WhatsApp interactions.

Key Features

  • Multi-Session Support: Manage multiple WhatsApp accounts simultaneously
  • Custom Commands: Create and manage custom bot commands
  • Real-time Logs: View detailed logs for each session
  • System Commands: Execute system commands directly from the web interface
  • User-friendly Interface: Easy-to-use dashboard for monitoring and managing all sessions

System Architecture

The WhatsApp Bot Manager consists of the following components:

  • Backend Server: Node.js server handling WhatsApp connections and API endpoints
  • Baileys Library: Core WhatsApp Web API implementation
  • Web Interface: Bootstrap-based responsive UI for managing the bot
  • Socket.IO: Real-time communication between the server and web interface
  • Command System: Modular system for handling WhatsApp commands

Getting Started

System Requirements

  • Node.js 14.x or higher
  • Modern web browser (Chrome, Firefox, Edge)
  • Internet connection
  • WhatsApp account(s)

Installation

To install and run the WhatsApp Bot Manager:

  1. Clone the repository or download the source code
  2. Navigate to the project directory
  3. Install dependencies: 
    npm install
  4. Start the server: 
    node index.js
  5. Open your browser and navigate to: 
    http://localhost:3000

First-time Setup

When you first start the WhatsApp Bot Manager, you'll need to add at least one WhatsApp session. Follow these steps:

  1. Click on "Add Session" in the sidebar
  2. Enter your phone number (with country code, e.g., 918XXXXXXXXX)
  3. Provide a session name (optional)
  4. Click "Start Session"
  5. Scan the QR code with your WhatsApp app to link your account

Once connected, you'll be redirected to the dashboard where you can see your active session.

Managing Sessions

Adding a New Session

To add a new WhatsApp session:

  1. Navigate to the "Add Session" page from the sidebar
  2. Enter your phone number with country code (e.g., 918XXXXXXXXX)
  3. Provide an optional session name for easy identification
  4. Click "Start Session"
  5. Scan the QR code with your WhatsApp app

Viewing Active Sessions

The Dashboard displays all your active WhatsApp sessions. For each session, you can see:

  • Phone number
  • Connection status (Connected, Connecting, Disconnected)
  • Session ID
  • Start time
  • Message count

Stopping a Session

To stop an active WhatsApp session:

  1. Go to the Dashboard
  2. Find the session you want to stop
  3. Click the "Stop Session" button
  4. Confirm the action when prompted

Note: Stopping a session will disconnect your WhatsApp account from the bot. You'll need to scan the QR code again to reconnect.

Session States

A WhatsApp session can be in one of the following states:

  • Connected - Successfully connected to WhatsApp
  • Connecting - In the process of connecting
  • Disconnected - Lost connection to WhatsApp

Bot Commands

Understanding Commands

Commands are JavaScript modules that define how the bot responds to specific messages. When a user sends a message that starts with a command prefix (e.g., "!command"), the bot executes the corresponding command module.

Available Commands

The WhatsApp Bot Manager comes with several built-in commands:

  • arise - Wake up the bot
  • auto_chat - Automated chat responses
  • ffinfo - Get file information
  • image_gen - Generate images
  • insta_reel - Download Instagram reels
  • memory - Memory management
  • system_call - Execute system commands
  • timer_off - Set timers
  • yt_music - Download YouTube music
  • yt_video - Download YouTube videos

Creating Custom Commands

To create a new custom command:

  1. Navigate to the "Manage Commands" page
  2. Click "Create New Command"
  3. Enter a name for your command (without spaces or special characters)
  4. Write your command code in the editor
  5. Click "Save Command"

Command Structure

A basic command module has the following structure:

module.exports = {
    name: "commandname",
    description: "Command description",
    execute: async (sock, msg, args) => {
        // Command logic here
        // sock - WhatsApp socket connection
        // msg - Message object
        // args - Command arguments
        
        // Example: Reply to the message
        await sock.sendMessage(msg.key.remoteJid, { text: "Hello!" }, { quoted: msg });
    }
};

Editing Commands

To edit an existing command:

  1. Go to the "Manage Commands" page
  2. Select the command you want to edit from the list
  3. Modify the code in the editor
  4. Click "Save Command"

Deleting Commands

To delete a command:

  1. Navigate to the "Manage Commands" page
  2. Select the command you want to delete
  3. Click the "Delete" button
  4. Confirm the deletion when prompted

Viewing Logs

Understanding Logs

The WhatsApp Bot Manager provides comprehensive logging for all sessions. Logs help you monitor bot activity, troubleshoot issues, and understand what's happening with your WhatsApp sessions.

Accessing Logs

There are two ways to access logs:

  1. From the Dashboard: Click the "View Logs" button for a specific session
  2. From the Sidebar: Click on "Logs" to view logs for all sessions

Log Types

Logs are color-coded by type for easy identification:

  • Info - General information about the session
  • Error - Error messages and exceptions
  • Warning - Warning messages
  • Incoming - Incoming messages
  • Outgoing - Outgoing messages

Filtering Logs

You can filter logs by:

  • Session: Select a specific session from the dropdown
  • Log Type: Filter by info, error, warning, incoming, or outgoing
  • Time Range: View logs from a specific time period

Clearing Logs

To clear the log display, click the "Clear Logs" button in the log controls section. This only clears the display and does not delete the actual log files.

System Commands

What are System Commands?

The System Commands feature allows you to execute various system operations directly from the web interface. This includes package installation, system information retrieval, network diagnostics, and custom commands.

Command Types

The System Commands page supports four types of commands:

  • Custom Commands: Run any custom command (with security restrictions)
  • Package Installation: Install packages using npm, pip, chocolatey, or winget
  • System Information: Get information about your system
  • Network Tools: Run network diagnostics and tests

Package Installation

To install a package:

  1. Select "Package Installation" from the command type dropdown
  2. Choose a package manager (npm, pip, chocolatey, winget)
  3. Enter the package name
  4. Check "Install Globally" if needed
  5. Click "Execute Command"

Note: All package installations automatically answer "yes" to any prompts.

System Information Commands

Available system information commands:

  • systeminfo: Display detailed system information
  • dir: List directory contents
  • tasklist: Show running processes
  • ver: Display OS version
  • echo %PATH%: Show PATH environment variable

Network Tools

Available network tools:

  • ipconfig: Display network configuration
  • ping: Test connectivity to a host
  • netstat: Show network statistics
  • tracert: Trace route to a destination

Security Restrictions

For security reasons, certain potentially dangerous commands are blocked, including:

  • Commands that could delete files or directories
  • Commands that could format drives
  • Commands that could modify system files
  • Commands that could shut down the system

Troubleshooting

Common Issues and Solutions

Problem: Unable to connect to WhatsApp

Solutions:

  • Ensure your internet connection is stable
  • Verify that your WhatsApp account is active on your phone
  • Try logging out and scanning the QR code again
  • Check if your WhatsApp version is up to date
  • Restart the bot server

Problem: QR code not appearing or not scanning

Solutions:

  • Refresh the page to generate a new QR code
  • Ensure your phone camera is working properly
  • Try using WhatsApp Web's "Scan QR Code" option directly
  • Clear browser cache and cookies
  • Restart the bot server

Problem: Commands not executing or responding

Solutions:

  • Check the logs for any error messages
  • Verify that the command syntax is correct
  • Ensure the command file exists and is properly formatted
  • Try reloading the commands by saving them again
  • Restart the bot server

Problem: System commands failing or not executing

Solutions:

  • Verify that the command is allowed by the security restrictions
  • Check if the required tools are installed on your system
  • Ensure you have the necessary permissions to execute the command
  • Try running the command with administrator privileges
  • Check the command output for specific error messages
